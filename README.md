# Padfak University Entrance Exam Simulator

این پروژه یک سایت استاتیک برای شبیه‌سازی آزمون ورودی دانشگاه‌های روسیه / پادفک است.

## امکانات

- صفحه لندینگ رسمی و مناسب آزمون ورودی دانشگاه
- انتخاب موضوع آزمون: روسی، شیمی، زیست
- انتخاب زبان سؤال‌ها برای شیمی و زیست: انگلیسی یا روسی
- ۱۰ سؤال نمونه برای هر بانک سؤال
- انتخاب تعداد سؤال بین ۰ تا ۱۰
- تنظیم مدت آزمون برحسب دقیقه
- مقدار پیش‌فرض تعداد سؤال: ۵ سؤال
- مقدار پیش‌فرض زمان آزمون: ۳۰ دقیقه
- تایمر آزمون با نمایش دقیقه و ثانیه
- انتخاب گزینه و امکان تغییر پاسخ
- ثبت خودکار آزمون بعد از پایان زمان
- نمایش نتیجه نهایی، درصد، پاسخ‌های درست/غلط/بدون پاسخ
- نمایش پاسخ صحیح و توضیح کوتاه برای هر سؤال
- طراحی واکنش‌گرا برای موبایل و دسکتاپ

## روش اجرا

فایل `index.html` را با مرورگر باز کنید.

نیازی به نصب پکیج یا اجرای سرور نیست.

## ساختار فایل‌ها

```text
padfak-exam-simulator/
├── index.html
├── css/
│   └── styles.css
└── js/
    ├── app.js
    └── questions.js
```

## ویرایش سؤال‌ها

برای تغییر یا اضافه کردن سؤال‌ها، فایل زیر را ویرایش کنید:

```text
js/questions.js
```
